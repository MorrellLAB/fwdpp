Detailed Description

Base class for mutations At minimum, a mutation must contain a position and a count in the population. You can derive from this class, for instance to add selection coefficients, counts in different sexes, etc.

Note
See Mutation policies in Tutorial 1: Policies in fwdpp for more detail on how to extend this type
Examples:
diploid.cc, diploid_binaryIO.cc, diploid_binaryIO_ind.cc, diploid_ind.cc, diploid_ind_2locus.cc, diploid_twopop_mig.cc, and TFL2013.cc.
