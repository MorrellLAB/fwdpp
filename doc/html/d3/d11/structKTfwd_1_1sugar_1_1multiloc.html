Detailed Description

template<typename mutation_type, typename mcont, typename gcont, typename dipvector, typename mvector, typename ftvector, typename lookup_table_type>
struct KTfwd::sugar::multiloc< mutation_type, mcont, gcont, dipvector, mvector, ftvector, lookup_table_type >

Abstraction of what is needed to simulate a multilocus simulation using an individual-based sampler from fwdpp.

All that is missing is the mutation_type and the container types.

Does not allow copy construction/assignment!

See Tutorial 5: the "sugar" layer on top of fwdpp for rationale, etc.

Examples:
diploid_ind_2locus.cc.
