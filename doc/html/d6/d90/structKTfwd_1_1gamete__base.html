Detailed Description

template<typename mut_type, typename list_type = std::list<mut_type>>
struct KTfwd::gamete_base< mut_type, list_type >

Base class for gametes. A gamete is a container of pointers (iterators) to mutations + a count in the population.

Note
The typical use of this class is simply to define your mutation type (see TotMut) and then use a typedef to define your gamete type in the simulations:
using gamete_t = KTfwd::gamete_base<mutation_type>
See Tutorial 1: Policies in fwdpp for examples of this.
Examples:
diploid_fixed_sh_ind_lambda.cc, and pfix.cc.
