Detailed Description

These functions pull a sample of simulated individuals from the population. You may only use them on the output of individual-based simulations (gamete-based simulations never explicitly store individuals).

The individuals are sampled uniformly, and with replacement, with no regard for their fitness.

The object passed to these functions is a container of diploids.

For these functions, must be an even number, as it represents the number of alleles to sample (twice the number of individuals).

The return values of these functions store individuals in the order that they were sampled.
