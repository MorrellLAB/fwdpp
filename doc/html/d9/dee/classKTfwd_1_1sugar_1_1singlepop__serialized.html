Detailed Description

template<typename mutation_type, typename mwriter, typename mreader, typename mlist, typename glist, typename dipvector, typename mvector, typename ftvector, typename lookup_table_type, typename dip_reader_t = KTfwd::diploidIOplaceholder, typename dip_writer_t = KTfwd::diploidIOplaceholder>
class KTfwd::sugar::singlepop_serialized< mutation_type, mwriter, mreader, mlist, glist, dipvector, mvector, ftvector, lookup_table_type, dip_reader_t, dip_writer_t >

Abstraction of what is needed to simulate a single population using an individual-based sampler from fwdpp.

Allows copy/construction/assignment via deep copy.

See Tutorial 5: the "sugar" layer on top of fwdpp for rationale, etc.
