Create admixed population.

Create an admixed population with a specified amount of ancestry from each parental deme.

Parameters
rA pointer to a gsl_rng object
mutationsContainer of mutations
mcountsContainer of mutation counts
gametesContainer of gametes
diploidsContainer of diploids in a metapopulation (vector of vectors of diploids).
iThe index of one parental deme
jThe index of the other parental deme
piThe fraction of ancestors in the admixed deme coming from deme i
N_newThe size of the new, admixed deme
replacementWhether or not to sample parents with replacement from their respective demes.

The behavior of this function depends on the value of replacement. When replacement is false, the size of the new deme cannot be so large that it requires all of demes i and/or j. When replacement is true, there is no limit on the value of N_new.

Returns
Error value -1 if i or j >= diploids.size(), -1 if replacement is true and the new deme size is too large, 1 if pi < 0. or pi >= 1., 0 otherwise (meaning successful admixture).

Split deme into two.

Take deme i and create a new deme of size N_new from it. Deme i's size is reduced N_new individuals.

Parameters
rA pointer to a gsl_rng object
mutationsContainer of mutations
mcountsContainer of mutation counts
gametesContainer of gametes
diploidsContainer of diploids in a metapopulation (vector of vectors of diploids).
iThe index of the deme to split.
N_newThe population size of the new deme.
replacementSample individuals with replacement or not.

The behavior of this function depends on the value of replacement. If replacement is false, then N_new individuals are selected at random (w/o replacement), removed from deme i, and added into the new deme. For this case, N_new must also be <= diploids[i].size()

When replacement is true, then deme i is first copied to a location outside of diploids. Then, N_new individuals are randomly chosen (with replacement) from the copy of deme i and added into the new deme. Likewise, diploids[i].size()-N_new individuals are randomly chosen (with replacement) from the copy of deme i and used to re-populate deme i.

Returns
Error code -1 if i>=diploids.size(), 1 if (!replacement&&(N_new >= diploids[i].size())), 0 otherwise (meaning successful split).
Note
Gamete and mutation counts are updated only when sampling with replacement. Counts cannot change if sampling without replacement.
