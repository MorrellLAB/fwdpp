14  \brief Mutation type allowing arbitray number of "s,h" pairs representing effect sizes,dominance.
15  \ingroup sugar
16 
17  This type represents a mutation with a number of 's' and 'h' parameters that are fixed
18  at compile time.
19 
20  For example, generalmut<2> could represent separate effects on fitness and a trait, while
21  generalmut<4> could represent separate effects on fitness and on a trait in males and females.
22 
23  Other example use cases could be generalmut<N> representing fitness/dominance value pairs in
24  different 'patches' of an environment.
25 
26  The advanatage of this type is that std::array<double,N> is a very fast type (often faster
27  than a std::vector<double> of the same size). Further, this type can be filled via compile-time
28  meta-programming techniques.
29 
30  The disadvantage of this type is that it is not easy to have the number of 's,h' pairs be flexible
31  at run-time. For that use case, see KTfwd::generalmut_vec, which uses std::vector<double> instead of
32  std::array to store the 's,h' values.
33  */
