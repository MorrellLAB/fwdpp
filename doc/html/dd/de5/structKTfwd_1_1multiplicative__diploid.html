Detailed Description

Multiplicative fitness across sites.

Parameters
g1A gamete
g2A gamete
mutationContainer of mutations
scalingFitnesses are 1, 1+h*s, 1+scaling*s, for AA,Aa,aa, resp. This parameter lets you make sure your simulation is on the same scale as various formula in the literature
Returns
Multiplicative fitness across sites.

bneck_selection_ind.cc, diploid_binaryIO_ind.cc, diploid_fixed_sh_ind.cc, diploid_ind.cc, and migsel_ind.cc.
