12 /*!
13  \file gamete_cleaner.hpp
14 
15  This file handles the "pruning" of fixations from gametes.
16 
17  Whether or not any pruning happens depends on policies.
18 
19  Version 0.4.7 of the library made a major change resulting in improved
20  run-time performance, especially for low mutation rates.
21 
22  Consider the case of neutral mutations occurring at rate mu (per site,
23  per generation). The rate of fixation is also mu. Thus, when mu is low,
24  the rate of fixation is low, and the expected time in generations between
25  fixations is 1/mu. In other words, calling gamete_cleaner every generation
26  is an unnecessary run-time cost. Now, we check if any fixations exist, and return
27  if there are none. We check separately for neutral- and non-neutral fixations.
28 
29  Naively, we could simply scan the mutations container of the entire simulation for fixations.
30  But, we can do better than that because:
31 
32  1. Fixations are, by definition, found in all gametes. Thus, it suffices to check the
33  first (extant) gamete.
34 
35  2. When we find the first fixation in a gamete, it is by definition the fixation with the smallest
36  position, out of all fixations. We can store its value and use it to look up the first fixation
37  in all of the remaining gametes. Searching for the first fixation in this way avoids cache misses that
38  are unavoidable when we do out-of-order lookups in "mcounts" for the remaining fixations.
39 */

216  template<typename gcont_t,
217  typename fixation_finder>
218  std::pair<bool,bool> fixation_finder_search_all(const gcont_t & gametes,
219  const fixation_finder & ff) noexcept
220  /*!
221  Determines whether a fixation exists in ANY gamete.
222  */
223  {
224  bool neutral=false,selected=false;
225  for(const auto & g : gametes)
226  {
227  if(!neutral)
228  {
229  auto itr = ff(g.mutations);
230  if(itr != g.mutations.cend())
231  {
232  neutral=true;
233  }
234  }
235  if(!selected)
236  {
237  auto itr = ff(g.smutations);
238  if(itr != g.smutations.cend())
239  {
240  selected=true;
241  }
242  }
243  if(neutral && selected) return std::make_pair(neutral,selected);
244  }
245  return std::make_pair(neutral,selected);
