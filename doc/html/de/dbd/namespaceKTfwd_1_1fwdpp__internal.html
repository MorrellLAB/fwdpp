Mechanics of adding a new mutation. We do this odd choosing between find_if and upper_bound b/c of the way the GCC STL is implemented.

Objectively, upper_bound should always be preferred, but that is not the case in practice because the GNU STL implements find_if in a very clever way for small containers.

The mechanics of recombination for multilocus simulations The mechanics of recombination for multilocus simulations.

Parameters
rA GSL random number generator
recA policy that affects recombination withing a locus
bwA policy that returns the number of recombination events between loci. The return value must be unsigned int, and what really matters is if that value is odd or even.
r_between_lociA const array of doubles corresponding to the recombination rates between loci. For k loci, this vector must be k-1 doubles long. Further, the i-th value must correspond to the recombination rate between loci i-1 and i.
iA dummy index set equal to which locus we are processing. Start counting from 0.
parental_gamete_1An iterator to the first parental gamete
parental_gamete_2An iterator to the second parental gamete
g1If true, then parental_gamete_1 is inherited by the descendant.
LOShould be true if the last recombination even within loci resulted in an odd number of crossovers.
